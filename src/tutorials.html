Let's talk about SASS

We are going to talk about Sass which is a CSS preprocesor that help us to give style to our markup but with extra abilties that make our code more elegant powerfull and easier to write.


Sass includes the following features:

  • We can declare variables in a similar way than we do in javascript, so we can for example assign a color to a variable and then reuse the same variable in different parts of the code, this give us the ability to change the color in just one place and see the changes in all the places where we used that variable.
